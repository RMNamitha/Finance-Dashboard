#  Finance Analytics Dashboard

An interactive **Finance Analytics Dashboard** built using React that helps users visualize financial data, track spending patterns, and gain insights into income, expenses, and overall balance.

This project demonstrates how modern web technologies can be used to create **data-driven dashboards with meaningful financial insights and visualizations.**

---

#  Project Overview

The Finance Analytics Dashboard allows users to:

- Track **income and expenses**
- Visualize **balance trends over time**
- Analyze **spending by category**
- View a **transaction history table**
- Get **automatic financial insights**

The dashboard provides a **clean and intuitive interface** that helps users understand their financial activity through charts and summaries.

---

#  Features

### 📊 Summary Cards
Quick overview of:
- Total Income
- Total Expenses
- Current Balance

### 📈 Balance Trend Chart
A line chart that shows how the **financial balance changes over time** based on transactions.

### 🥧 Spending Category Analysis
A pie chart that displays **how expenses are distributed across categories**.

### 📋 Transaction Table
Displays all transactions including:
- Date
- Category
- Amount
- Transaction Type

### 🔐 Role Selection
Users can switch roles between:
- Viewer
- Admin

### 💡 Financial Insights
Automatically generated insights such as:
- Total income vs expenses
- Net savings
- Basic financial advice based on spending patterns

# ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/yourusername/finance-dashboard.git
```

### 2️⃣ Navigate to the project folder

```
cd finance-dashboard
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Start the development server

```
npm run dev
```

### 5️⃣ Open in browser

```
http://localhost:5173
```
